Let's stop here for a second. You may noticed that we've built this using:

  • Bootstrap container to center section on the screen
  • Some headings and paragraphs utilizing text classes
  • Bootstrap grid with two unequal columns to achieve proportion and mobile responsiveness
  • Image and icons

That's easy, isn't it?

Another point - as you noticed we are using section tag. It doesn't affect our page at all, it's like an empty div but it helps us to browser through the code much easier.
